# Claude Code Start Prompt

Copy this prompt into Claude Code immediately after opening the project.

```text
Ты работаешь над сайтом-портфолио UX/UI дизайнера Roman Zhalyalov.

Сначала ничего не редактируй. Прочитай:
1. AGENTS.md
2. CLAUDE.md
3. docs/handoff.md

Потом, если задача будет касаться проектов, кейсов, текстов или изображений, прочитай src/data/projects.ts.
Если задача будет касаться сетки, типографики, адаптива или визуальной полировки, прочитай src/styles/global.css.

Важно:
- Figma является важным источником, но не абсолютной истиной.
- В проекте уже есть осознанные отклонения от Figma после моего фидбека.
- Не возвращай сайт к Figma автоматически, если AGENTS.md, CLAUDE.md или docs/handoff.md говорят, что текущее отличие утверждено.
- Если нужно сравнить с Figma, используй Figma MCP/design tools, если они подключены в этом Claude Code окружении.
- Если Figma tools не подключены, честно скажи, что не видишь Figma, и попроси у меня скриншот, node link или экспортированный ассет.
- Не делай вид, что посмотрел Figma, если ты смотрел только локальные файлы.
- Не добавляй Tailwind.
- Не добавляй зависимость от Vercel.
- Не переписывай структуру проекта без необходимости.
- Перед финальным ответом запускай проверку через /portfolio-verify или команды build/check из AGENTS.md.

Особенно важно сохранить:
- текущий custom sidebar на главной: большой номер, название проекта, короткое описание;
- отсутствие бежевых/серых подписей поверх превью на главной;
- отсутствие старого Figma factoid layout с логотипом рядом с огромным номером;
- на страницах кейсов desktop screenshots без browser chrome;
- browser chrome означает декоративную верхнюю панель браузера: точки, адресную строку, фейковую рамку окна или header окна;
- общий радиус плиток/скриншотов --case-radius: 12px.

Теперь кратко перескажи, что ты понял:
1. стек проекта;
2. текущий статус;
3. что уже визуально утверждено;
4. где мы намеренно отошли от Figma;
5. что нельзя менять без моего подтверждения;
6. какие команды проверки надо запускать.

После пересказа остановись и жди мою конкретную задачу.
```
